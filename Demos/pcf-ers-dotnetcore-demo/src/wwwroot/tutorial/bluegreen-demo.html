For the purpose of this demo, we'll track the application name environment variable when starting the test.

  1. Deploy V2 (green) of the application
  2. Add the production route to application V2 (green)
  3. Ensure to control the load from V1 (blue) to V2 (green) by making sure you have more V1 (blue) application instance. For example, 2 x 1 application instances ensure V1 (blue) gets double the amount of requests.
  4. Decrease V1 (blue) instances, increase V2 (green) as confidence builds
  5. Remove the production route of V1 (blue).
